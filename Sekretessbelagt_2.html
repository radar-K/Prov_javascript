---------------------------------------------------------------------------------- I JavaScript är filter() en metod som används på arrayer för att skapa en ny array som innehåller endast de element som uppfyller ett visst villkor. map() används för att transformera varje element i en array genom att applicera en funktion på det. Resultatet blir en ny array där varje element har "mappats om" (dvs. förändrats) baserat på funktionen, men alla ursprungliga element är kvar. function filterAgeAndUpperCase(people) { return people .filter(person => person.age >= 18) // Filtrera ut personer som är 18 eller äldre .map(person => person.name.toUpperCase()); // Omvandla namnen till versaler } ---------------------------------------------------------------------------------- ---------------------------------------------------------------------------------- Man vill använda some() när man vill kontrollera om minst ett element i en array uppfyller ett visst villkor. ---------------------------------------------------------------------------------- ---------------------------------------------------------------------------------- Om du har en lista över resurser eller objekt och vill veta om någon av dem är i ett specifikt tillstånd (som tillgänglig eller aktiv), är some() effektivt. ---------------------------------------------------------------------------------- ---------------------------------------------------------------------------------- every() är en metod i JavaScript som används på arrayer för att kontrollera om alla element i en array uppfyller ett visst villkor. Om varje element uppfyller villkoret returnerar every() true; om minst ett element inte gör det, returnerar den false. Den är användbar när du vill säkerställa att en viss egenskap eller kriterium gäller för hela arrayen. Skillnaden mellan every() och some() * every() returnerar true endast om alla element uppfyller villkoret. * some() returnerar true om minst ett element uppfyller villkoret. ---------------------------------------------------------------------------------- ---------------------------------------------------------------------------------- Kontrollerar om en array innehåller ett visst värde och returnerar true eller false. Används för att snabbt kontrollera närvaron av ett värde i en array. ---------------------------------------------------------------------------------- ---------------------------------------------------------------------------------- Används för att reducera en array till ett enda värde, genom att applicera en funktion på varje element. array.reduce(callback(accumulator, currentValue, currentIndex, array), initialValue);
